Task B¶

I calculated the Shapley values using the following formula, as it is more efficient:

$$ \phi_j = \sum_{S \subseteq P \setminus \{j\}} \frac{|S|! \cdot (|P| - |S| - 1)!}{|P|!} \cdot (v(S \cup \{j\}) - v(S)) $$

The code for calculating the Shapley values is in the appendix and the results for the following players are presented below:

  • Player 'a': 25
  • Player 'b': 25
  • Player 'c': 50

I also double checked the results by calculating the Shapley values by hand, but even here we can perform a quick check by observing that the Shapley values correctly sum to v('abc') = 100 (efficiency) and players 'a' and 'b' have the same Shapley value (symmetry).

def generate_subsets(lst):
    """
    Generator function yielding all subsets of a given list (including the empty subset).
    :param lst: List of elements
    :return: Generator yielding all subsets of the given list
    """

    def generate_subset_recursive(current_subset, index):
        """
        Recursive function that generates all subsets of a given list (including the empty subset).
        :param current_subset: Current subset
        :param index: Current index in the list
        :return: Generator yielding all subsets of the given list
        """
        if index == len(lst):
            yield current_subset
            return

        # Include the current element in the subset
        yield from generate_subset_recursive(current_subset + lst[index], index + 1)

        # Exclude the current element from the subset
        yield from generate_subset_recursive(current_subset, index + 1)

    yield from generate_subset_recursive('', 0)

import math
from typing import Dict, List


def calculate_shapley_value(player_id: str, players_ids: List[str], coalitions_dict: Dict[str, int]) -> float:
    """
    Calculates the Shapley value for each player in a game with the given coalitions and their values.
    :param player_id: id of the player for which the Shapley value should be calculated
    :param players_ids: list of players ids
    :param coalitions_dict: dictionary of coalitions and their values (ids must be sorted)
    
    :return: shepley value for the player with the given id
    """
    subset_players_ids = players_ids.copy()
    subset_players_ids.remove(player_id)

    P = len(players_ids)
    P_factorial = math.factorial(P)

    shapley_value = 0
    for coalition in generate_subsets(subset_players_ids):
        S = len(coalition)
        previous_coalition_value = coalitions_dict[coalition]
        current_coalition_value = coalitions_dict[''.join(sorted(coalition + player_id))]
        shapley_value += (current_coalition_value - previous_coalition_value) * math.factorial(S) \
                         * math.factorial(P - S - 1)

    return shapley_value / P_factorial
